  • php-tick is a simple active record implementation with the least amount of fuss needed to make it work fast.
  • It has support for both SQL and noSQL databases.
  • It has no support for foreign key relations.

INSTALLATION

php-tick is PSR-4 compliant and easily installed with Composer

REQUIREMENTS

  • Php version 5.3.3< build with the following extension: pcre, SPL (default build-in in most distributions)
  • The SOLR storage engine requires the PECL SOLR extension.
  • The Mongo storage engine requires the PECL Mongo extension.
